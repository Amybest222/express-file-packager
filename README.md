# Express File Packager

> Package your Adobe Express files for instant handoff — transparent images, print-ready PDF, and font list in one click.

---

## What it does

Express File Packager bundles everything you need from your Adobe Express document into one tidy package. It exports your images with transparent backgrounds, generates a print-ready PDF of your project, and produces a font list of every typeface used in your document — so you and your collaborators always know exactly what fonts are in play. No manual exporting, no missing files. Just click, package, and go.

---

## How to use it

1. Open your document in [Adobe Express](https://www.adobe.com/express/)
2. Click the **Add-ons** icon in the left rail
3. Search for **Express File Packager** and install it
4. Open the add-on panel and click **Package Files**
5. Choose which outputs you need — transparent images, font list, PDF, or all three
6. Download your packaged zip file

---

## Output files

| File | Description |
|------|-------------|
| `images/` | PNG exports with transparent backgrounds |
| `fonts.txt` | Full list of fonts used in your document |
| `export.pdf` | Print-ready PDF of your project |
